# ft_printf (42 School)

## 📋 Description
A custom implementation of the C standard library function `printf`. This project focuses on **variadic functions** and complex string parsing in C.

## 🌟 Features (Bonus 125/100)
This version is fully validated with all bonuses:
- **Mandatory**: `%c`, `%s`, `%p`, `%d`, `%i`, `%u`, `%x`, `%X`, `%%`.
- **Bonus Flags**: `-`, `0`, `.`, `#`, `+`, and `space`.

## 📂 Project Structure
- src/: Mandatory source files.
- _bonus/: Bonus source files and logic.

## 🛠️ Usage & Compilation
To compile the **mandatory** part:
```bash
make
To compile with all bonus flags:
```bash
make bonus
